# Miso Moon Café

Miso Moon Café is a fictional pastel Japanese-inspired café website built with HTML and CSS. The project features a dreamy restaurant concept with a hero section, about section, featured favorites, full menu cards, visit information, and a styled footer.

## Project Overview

This project was created as a beginner-friendly front-end portfolio project. The goal was to practice building a polished static webpage using semantic HTML, CSS layout, image cards, navigation links, and responsive design.

The café concept is inspired by cozy Japanese comfort food, pastel dessert shops, and whimsical moon-and-star themed food styling.

## Features

- Responsive one-page restaurant website
- Header with logo and navigation
- Hero section with featured café image
- About section with restaurant interior image
- Featured favorites card section
- Full menu section with image-based menu cards
- Visit information section
- Footer with social links
- Custom pastel color palette
- Accessible image alt text
